# Databricks Lakeflow Connect Data Ingestion
📌 Overview
This project demonstrates data ingestion using Databricks Lakeflow Connect with streaming and batch data pipelines. It focuses on loading raw files into Delta tables and tracking ingestion behavior in a medallion-style architecture.
